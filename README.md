# CPPProjects

This repository contains small beginner-level C++ programs created for practice and learning.

## About

This repo is a collection of simple console-based projects written in C++.  
It is mainly focused on improving fundamental programming skills such as:

- Variables and data types
- Conditionals (if/else, switch)
- Loops (for, while)
- Functions
- Basic problem-solving

These are not advanced projects—just practice exercises while learning C++.

## Projects Included

Here are some of the programs you may find in this repository:

- Tic Tac Toe (console version)
- Number Guessing Game
- Rock Paper Scissors
- (More small practice projects will be added over time)
