# CPPProjects

This repository contains small beginner-level C++ programs created for practice and learning.

## About

This repo is a collection of simple console-based projects written in C++.  
It is mainly focused on improving fundamental programming skills such as:

- Variables and data types
- Conditionals (if/else, switch)
- Loops (for, while)
- Functions
- Basic problem-solving

These are not advanced projects—just practice exercises while learning C++.

## Projects Included

Here are some of the programs you may find in this repository:

- Tic Tac Toe (console version)
- Number Guessing Game
- Rock Paper Scissors
- (More small practice projects will be added over time)

## How to Run

## Requirements
- A C++ compiler (GCC, Clang, or MSVC)
- Raylib (for graphical projects)
How to Run
- Clone the repository:
```bash
git clone https://github.com/your-username/CPPProjects.git
```
- Navigate to the project folder.
- Compile the desired project.

## Example for a console application:
```bash
g++ main.cpp -o program.exe
```
## Example for a Raylib project:
```bash
g++ main.cpp -o game.exe -lraylib -lopengl32 -lgdi32 -lwinmm
```
Run the executable.
